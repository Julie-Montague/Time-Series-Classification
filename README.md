# Time-Series-Classification-XAI
# OVERVIEW
This is an Explainable AI project that uses the the fingerMovements dataset to genrate predictions and uses several XAI methods to intepret the output . 
# DATASET
The dataset is can be obtained from https://timeseriesclassification.com/description.php?Dataset=FingerMovements
# Used Model:
  - Long-short Term Memory (LSTM) Model
# Used XAI METHODS:
  - Integrated Gradients
  - Saliency Maps
  - SHAP
